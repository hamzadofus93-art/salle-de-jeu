Phoenix Snooker Club

Le dashboard principal est maintenant servi par Express + Angular.

Cette ancienne page statique n'est plus l'entree de l'application. Pour charger le vrai dashboard, il faut lancer le serveur Node.js puis ouvrir l'application sur le port 3001.

Ouvrir l'application Lancer avec start-app.cmd
  • 1. Double-clique sur start-app.cmd a la racine du projet.
  • 2. Ou lance cmd /c npm --prefix server run start:full.
  • 3. Ensuite ouvre http://localhost:3001.
